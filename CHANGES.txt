FEATURE OVERVIEW
================================================================================

What it does:
  - Detects local products that no longer exist in remote Planet API
  - Takes configurable action (keep/draft/trash/delete)
  - Only affects products synced by this plugin
  - Manual WooCommerce products are never touched
  - All actions are logged for audit trail

HOW TO USE
================================================================================

1. Navigate to: Products â†’ Planet Sync 2.0

2. Scroll to Settings section

3. Find "Orphaned Products Action" dropdown

4. Select desired action:
   - Keep (Do Nothing) - Safe default, no changes
   - Set to Draft - Hide from store, keep for reference
   - Move to Trash - Soft delete, can restore within 30 days
   - Permanently Delete - Immediate removal (cannot undo!)

5. Click "Save Settings"

6. Run sync (manual or automatic)

7. Check "Orphaned Products" section to see results
